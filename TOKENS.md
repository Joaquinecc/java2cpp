# Tokens

### Operators

| Token 	| Meaning                 	| Java symbol 	| C++ symbol 	|
|-------	|-------------------------	|-------------	|------------	|
| LAND  	| Logical AND             	| &&          	| &&         	|
| LOR   	| Logical OR              	| \|\|        	| \|\|       	|
| GEQ   	| Greater or equal        	| >=          	| >=         	|
| LEQ   	| Less or equal           	| <=          	| <=         	|
| NOT   	| Logical negation        	| !           	| !          	|
| GT    	| Greater than            	| >           	| >          	|
| LT    	| Less than               	| <           	| <          	|
| NEQ   	| Not equal               	| !=          	| !=         	|
| DEQ   	| Double equal comparison 	| ==          	| ==         	|
| PLUS  	| Plus symbol             	| +           	| +          	|
| MINUS 	| Minus symbol            	| -           	| -          	|
| MUL   	| Multiplication symbol   	| *           	| *          	|
| DIV   	| Division symbol         	| /           	| /          	|
| MOD   	| Mod symbol              	| %           	| %          	|
| EX    	| Exponent symbol         	| ^           	| ^          	|
